Farm Simulator
By: Angel Azpeitia-Marquez, Ervin Martirosyan, Jason Thomas Zhang, Sydney Green

Genre:
Top down shooter, Horde Survival.

Movement/Controls:

D-Pad, WASD

W = Up

A = Left

S = Down

D = Right

Left-Click to Shoot

Goal: Survive from an assault of enemies, while protecting the livestocks.

Bugs:
